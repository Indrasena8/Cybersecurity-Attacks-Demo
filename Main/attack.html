Task 1: Understanding Packet Sniffing

This task introduces the Python module Scapy, a popular tool for sniffing and spoofing packets. You will use Scapy to sniff packets in this exercise.

What You Need to Do:

  1. Step 1: Review the sample code for sniffing in the Python program file named sniffer.py on Attacker host. This code sniffs the first 10 packets on Attacker host. Run this Python code with root privileges.
    sudo python3 sniffer.py
  2. Step 2: While the Python code is running on Attacker host, on Defender host execute the following command:
    3. ping -c 15 10.4.0.7

Task 2: Understanding Packet Spoofing

In this task, you will use Scapy to spoof a UDP packet.

What You Need to Do:

  1. Step 1: Locate the Python program file named spoofer.py on Attacker host. This file is designed to spoof a UDP packet with the following details:
    • Source IP: 10.4.0.7
    • Destination IP: 10.4.0.9
    • Source Port and Destination Port: 3345
    • Data Message: “spoofed payload”
  2. Step 2: On Defender host, run the following command:
    3. sudo tcpdump -n -i eth0 -c 100
  3. Step 3: Keep the tcpdump command running in both terminals, then execute the spoofer code
    sudo python3 spoofer.py
    file with root privileges.

Task 3: Performing a DoS Attack using hping

In this task, you will use hping3 to flood a host with packets, simulating a Denial of Service (DoS) attack.

What You Need to Do:

  1. Step 1: On the Attacker host, use hping3 to flood the Defender host with SYN packets. Run the following command:
    2. sudo hping3 -S -c 50 -V -p 80 10.4.0.9
    • Source IP: 10.4.0.7
    • Destination IP: 10.4.0.9
    • Destination Port: 80 (HTTP)
  2. Step 2: On Defender host, run the following command to capture incoming packets:
    3. sudo tcpdump -n -i eth0 -c 100
  3. Step 3: Keep the tcpdump command running on HostB and observe the packet flood for a short time.
